# HTML Encoder & Decoder Utility 🌐

A fast, lightweight, and completely client-side developer utility for instantly converting raw text and code snippets into safe HTML character entities, and vice versa. Built to provide a secure sandbox for input sanitization, preventing Cross-Site Scripting (XSS), and formatting code for web documentation with zero server latency.

## 🔗 Live Application

The secure, production-ready live tool can be accessed here:
👉 **[HTML Encoder & Decoder Utility](https://htmlenc.gt.tc/)**

## 🚀 Features

- **Bi-directional Conversion:** Seamlessly encode raw text to HTML entities or decode entities back to raw source code.
- **Comprehensive Handling:** Supports both standard named entities (e.g., `&lt;`, `&amp;`) and numerical character references.
- **100% Client-Side & Secure:** All transformations happen natively inside the browser. No input data is ever sent to a server, ensuring complete data isolation and privacy.
- **Minimalist UX:** A distraction-free, high-contrast interface designed for quick copying and pasting into your development workflow.
